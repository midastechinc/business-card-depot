# Business Card Depot Product Brief

## Purpose

Business Card Depot helps turn business cards and contact screenshots into usable phone contacts quickly.

## Intake sources

- Phone camera scan
- Imported image
- Screenshot from a website

## First release goal

The first useful release should:

1. Let the user capture or import an image
2. Run OCR on the image
3. Map the OCR output into contact fields
4. Let the user edit the fields
5. Save the contact to the phone

## Primary extracted fields

- Full name
- Company
- Job title
- Mobile phone
- Office phone
- Email
- Website
- Address
- Notes

## Nice-to-have later

- Duplicate detection
- CRM export
- Bulk import queue
- AI cleanup for messy OCR text
- Share contact as vCard

## Repo reminder

This app should have its own GitHub repository named `business-card-depot` or another final approved variant before active development continues.
